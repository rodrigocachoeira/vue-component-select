Documentação


Motivação

O componente select foi desenvolvido para facilitar o uso do componente de Seleção do HTML, o qual é de estrema facilidade de uso, porém ao atribuir algumas interações a ele o desenvolvimento pode ficar massivo. Alguns exemplos de utilização mais complexas são iterações entre elementos, estes sejam de seleção ou outros, como por exemplo, ao seleciona um registro em um componente de seleção um componente secundário é liberado e realiza uma requisição utilizando como base a infomrmação passado do primeiro componente como filtro de consulta.

Utilização

Vinculo com a instância primária do Vue JS

import SelectComponent from './SelectComponent.vue';
Vue.component('selectComponent', SelectComponent);

Propriedades

O componente é baseado totalmente em propriedades, para configurá-lo corretamente algumas propriedades são essenciais.

Path: string
path="/src/records/persons.json"
path="/records"

O atributo path indica o local que será requisitado as informações interadas no elemento HTML, as informações podem ser adquiridas de duas formas, requisitando um documento JSON ou requisitando um URL, esta que por sua vez retorna um JSON (GET, POST). O formato de resposta deve ser padrão Array[Object, Object] como no exemplo a seguir.

[{
  "id": 1,
  "name": "Rafaela",
  "age": 21
},
{
  "id": 2,
  "name": "Adriano",
  "age": 23
}]
E-text string | array
e-text="name"
e-text=['name', 'age']

O atributo e-text indica qual pripriedade do objeto a ser interado será apresentado dentro do select.

E-value string
e-value="id"

O atributo e-value indica qual pripriedade do objeto a ser interado será atribuído ao value do componente option.
